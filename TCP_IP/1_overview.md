## 概述

### 网络协议的分层

- 链路层：（设备驱动程序和网卡接口）处理与电缆的物理接口细节
- 网络层：处理分组在网络中的活动，有IP/ICMP/IGMP等协议
    + IP：网际协议
    + ICMP：互联网控制报文协议
    + IGMP：Internet组管理协议
- 运输层：为两台主机上的应用程序提供端到端的通信，如TCP/UDP
- 应用层：处理特定的应用程序细节。
    + Telnet远程登录
    + FTP文件传输协议
    + SMTP简单邮件传输协议
    + SNMP简单网络管理协议

### 互联网地址（IP地址）

32位组成，互联网上的每个接口都有一个唯一的IP地址。
InterNIC只分配网络号。

### 域名系统（DNS）

IP地址和主机名之间的映射。
